# Classes and Students

Organize students into classes for easier management.

## Creating a Class

1. **Dashboard → Classes**
2. Click **+ New Class**
3. Give it a name (e.g., "CS101 Fall 2024")
4. Copy the **invite code**

## Student Enrollment

Share the invite code with students. They:
1. Sign in to Eduskript
2. Enter the invite code
3. Join your class

Students appear in your class roster.

## Class Settings

| Setting | Description |
|---------|-------------|
| **Name** | Display name for the class |
| **Invite code** | Unique code students use to join |
| **Allow anonymous** | Let unauthenticated users access class content |

## Student Privacy

Eduskript protects student privacy:

- Students sign in via OAuth (Google, GitHub, etc.)
- No email addresses stored for students
- Students identified by pseudonym hash
- You see student activity, not personal details

## What You Can See

For enrolled students:
- Which pages they've visited
- Code they've written in editors
- Annotations they've made (if enabled)

## Pre-Authorized Students

Import a class roster before students enroll:

1. Create a CSV with student identifiers
2. Upload via **Class Settings → Import**
3. Students matching those identifiers auto-enroll

Useful for institutional integrations.

## Tips

- Create one class per course section
- Regenerate invite codes if leaked
- Archive old classes at semester end
